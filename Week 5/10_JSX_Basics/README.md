# JSX Basics

## Concept Overview
JSX stands for JavaScript XML. It is a syntax extension for React that allows developers to write HTML-like markup directly inside JavaScript files. While it looks like HTML, it is completely converted into standard JavaScript before running in the browser. Using JSX is not strictly required in React, but it makes the code significantly easier to write, read, and maintain compared to manually creating DOM nodes.

## What this Code Demonstrates
This project clearly illustrates what JSX does by comparing two identical pieces of UI. The first piece is built manually using the `React.createElement()` function, which is the underlying API React uses to build the virtual DOM. The second piece is built using JSX syntax. They both render exactly the same output, but the JSX version is visibly much closer to standard HTML, demonstrating why JSX is universally adopted in the React ecosystem.
